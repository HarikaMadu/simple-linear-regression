# Student Marks Prediction using Simple Linear Regression

## Overview

This project uses Simple Linear Regression to predict a student's marks based on the number of study hours. The model is trained using a sample dataset containing study hours and corresponding marks.

## Objective

The main objective of this project is to understand and implement Simple Linear Regression, a supervised machine learning algorithm used for predicting continuous values.

## Features

- Predicts student marks based on study hours
- Uses Simple Linear Regression algorithm
- Visualizes data using a regression line
- Takes user input for prediction
- Beginner-friendly machine learning project

## Technologies Used

- Python
- Pandas
- Scikit-Learn
- Matplotlib

## Dataset

The dataset contains:

| Study Hours | Marks |
|------------|--------|
| 1 | 20 |
| 2 | 30 |
| 3 | 35 |
| 4 | 50 |
| 5 | 55 |
| 6 | 65 |
| 7 | 70 |
| 8 | 85 |

## Working Process

1. Load the dataset.
2. Separate independent and dependent variables.
3. Train a Simple Linear Regression model.
4. Accept study hours as user input.
5. Predict marks based on the trained model.
6. Display the regression graph.

## Sample Output

```
Enter study hours: 5
Predicted Marks: 54.40
```

## Visualization

The graph displays:
- Data points representing study hours and marks
- Regression line showing the relationship between variables
